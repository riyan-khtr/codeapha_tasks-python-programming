# Streamlit Stock Portfolio Tracker (Python)

## Overview
This project is a simple web application built using Streamlit that helps users track their stock portfolio. The user can enter stock names and quantities, and the app calculates the total investment using fixed prices. It also displays the portfolio in a table and visualizes the distribution using charts.

## Objective
To build an interactive web app using Python and Streamlit while practicing data handling, visualization, and user input.

## Concepts Used
- Streamlit for web app interface
- Pandas for data handling
- Matplotlib for charts
- File handling to save portfolio data
- Basic Python logic and user input

## Project Files
- `app.py` – Main Streamlit application
- `requirements.txt` – Required libraries
